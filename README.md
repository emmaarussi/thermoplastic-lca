# Thermoplastic Life Cycle Assessment Tool

## Description
The Thermoplastic Life Cycle Assessment (LCA) tool is designed to evaluate the environmental impact of thermoplastic and composite materials throughout their lifecycle. This tool provides a comprehensive analysis by integrating various data sources and methodologies.
